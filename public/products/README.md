# Product photos

Drop a real photo for each product here, using the **exact file name** below.
As soon as a file exists, the landing automatically shows it instead of the
flat illustration (the fallback). Recommended: square images (e.g. 600×600),
`.jpg`, product centered on a clean/white background, < 200 KB each.

If you use a different extension (e.g. `.png`/`.webp`), update the matching
`image` path in `lib/products.ts`.
